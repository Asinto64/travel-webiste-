# Punjab Rides 🚕✨

A premium, cinematic single-page web experience for a Punjab-focused cab and ride-booking service. Designed with a dark-luxury, editorial aesthetic, this project emphasizes smooth user interactions, atmospheric high-resolution imagery, and modern web design principles.

## 🌟 Features

- **Cinematic Aesthetic**: Deep navy backgrounds paired with warm gold and saffron accents. Subdued noise filters and ambient background blobs create a glowing, luxurious atmosphere.
- **Smooth Scroll & Animations**: Integrated **Lenis** for buttery smooth inertia scrolling and **GSAP (ScrollTrigger)** for high-performance scroll-linked fade-ups, parallax layers, and staggered elements.
- **Hardware-Accelerated 3D Effects**: Interactive elements like destination cards and the vehicle fleet grid utilize 3D tilt transformations and glowing hover states without sacrificing frame rates.
- **Fully Responsive Layout**: An elegant, full-screen frosted glass mobile navigation menu and swipe-friendly native carousels for smaller screens. 
- **Custom AI Imagery**: Includes 6 unique 8k, photorealistic AI-generated images capturing the essence of Punjab (from the Golden Temple to the Pathankot mountains).
- **Custom Cursor**: A glowing dot with a lagging translucent ring that expands over interactive elements (disabled safely on touch devices).

## 🛠️ Tech Stack

- **HTML5**: Semantic document structure.
- **CSS3**: Vanilla CSS variables, glassmorphism filters, responsive grids, and highly customized animations. 
- **Vanilla JavaScript**: DOM manipulation, custom cursor logic, and animation loop syncing.
- **GSAP & Lenis**: Loaded securely via CDN for animation and scroll management.

## 🚀 How to Run

Because this project is built entirely with vanilla web technologies, there is no build step required! 

1. Clone the repository:
   ```bash
   git clone https://github.com/Asinto64/travel-webiste-.git
   ```
2. Navigate to the project directory.
3. Simply double-click the `index.html` file to open it in any modern web browser (Chrome, Edge, Safari, Firefox).
